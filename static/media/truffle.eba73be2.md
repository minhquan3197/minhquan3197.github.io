# Truffle Cheatsheet

- Init truffle

```bash
truffle init
```

- Create contract

```bash
truffle create contract <name_contract>
```

- Compile contract

```bash
truffle comile
```

- Migration - deploy contract

```bash
truffle migrate
```

- Create test contract

```bash
truffle create test SimpleContract
```

- Run test

```bash
truffle test
```
